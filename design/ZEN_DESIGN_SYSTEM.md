# Design System: Bright Zen Minimalist
## Agentic AI Start-Up: Engineer Your AI-Native Business
**Updated:** 2026-05-18

## Philosophy
Think Notion meets Linear meets a Japanese design studio. Calm. Focused. Every element earns its place. The interface gets out of the way so the student can think.

## Color Palette
- Background: warm off-white (#FAFAF8)
- Primary text: near-black (#1A1A1A)
- Secondary text: medium gray (#6B7280)
- Accent: sage green (#4A7C59) — used sparingly for active states, buttons, progress
- Borders: very light gray (#E5E7EB)
- Cards: white with soft shadow (0 2px 8px rgba(0,0,0,0.06))
- No gradients. No dark backgrounds. No neon.

## Agent Avatar Colors (muted, soft)
- Arch: warm clay (#C4A882)
- Scout: sky blue (#A8C4C4)
- Reed: soft amber (#C4B882)
- Vera: dusty rose (#C4A8A8)
- Mo: slate (#A8B4C4)
- Sage: sage green (#A8C4A8)

## Typography
- Font: Inter or Geist
- Headings: weight 300–400 (light, not heavy)
- Body: 16px minimum, generous line spacing
- Letter spacing: slightly loose

## Layout
- Max content width: 680px centered
- Generous whitespace — content breathes
- Cards: 8px border radius, subtle shadow
- No dividers — use whitespace instead

## Navigation
- Slim left sidebar: Home, Progress, Agents only
- Active state: thin sage green left border
- App name: small, top-left, light gray

## Buttons
- Primary: sage green (#4A7C59), white text, 6px radius, flat
- Secondary: white bg, light gray border, dark text
- Hover: slight darken, no dramatic effects

## Chat Interface
- Student messages: white bubble, light gray border, right-aligned
- Agent messages: light tinted bg (agent color), left-aligned
- Input: clean white, minimal border, sage green send button
